Приложение разрабатывалось с применением Angular 5, Bootstrap 4, Angular Material и Firebase

У приложения есть два варианта использования (их можно выбрать справа ввеху в меню с тремя точками):

  • "Словарь" используется для внесения слов и переводов в базу данных
  • "Квиз" используется для проверки собственных знаний на основе внесенных в базу данных слов и переводов

Планируется доделать

  • Авторизацию через соцсети или почту
  • Персональные БД со словами
  • Мультиязычность
